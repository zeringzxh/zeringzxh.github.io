0x01 GSIL介绍

GSIL全称GitHub Sensitive Information Leakage（GitHub敏感信息泄露监控）。
功能：

  • 根据rules.gsil规则匹配GitHub内容
  • 通过邮件告警
  • 设置计划任务，定期扫描

安装

使用python3进行安装,由于同时存在python2，因此使用python3，pip3来调用。
